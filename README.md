# Depot d'add-on Home Assistant : Automations Graph

Ce depot contient un seul add-on : **automation_graph** (voir
`automation_graph/DOCS.md` pour l'installation et l'utilisation).

## Installation rapide (add-on local)

Copie le dossier `automation_graph/` dans `/addons/` sur ton serveur Home
Assistant (Samba, SSH, ou l'add-on Studio Code Server), puis dans
**Parametres > Modules complementaires > Boutique > (⋮) > Actualiser**.

## Installation en tant que depot (optionnel)

Si tu preferes le gerer comme un vrai depot (mises a jour via Git) :
publie ce dossier tel quel sur GitHub, puis dans Home Assistant va dans
**Parametres > Modules complementaires > Boutique > (⋮) > Depots** et ajoute
l'URL de ton depot.
